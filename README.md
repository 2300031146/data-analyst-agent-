## 📌 Project Overview

AI Data Analyst Agent is a web-based interface designed to simulate an end-to-end data analysis pipeline.
It allows users to upload CSV files and visualize a complete workflow including data ingestion, SQL querying, Python-based analysis, dashboards, and report generation.

## ⚙️ Features

* 📂 CSV Upload Interface
* 🧠 SQL Query Simulation
* 🐍 Python EDA Workflow
* 📊 Dashboard Visualization UI
* 📈 Automated Insights (UI-based)
* 📑 PowerPoint Report Generator (UI)
* 🔗 GitHub Report Integration

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript (UI simulation)
* GitHub Pages (Deployment)
